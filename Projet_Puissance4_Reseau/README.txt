1 COMPILER
	Lancer le Makefile : 
	$make


2 EXECUTER
	A- Dans un terminal :
		Lancer le serveur :
		$./server 8080 &

	B- Dans un terminal :
		Lancer le client numero 1 :
		$./client localhost 8080

	C- Dans un terminal :
		Lancer le client numero 2 :
		$./client localhost 8080
					
	D- Jouer dans les deux clients.
